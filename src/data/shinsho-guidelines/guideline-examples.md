# 信書に関する基本的な考え方

# (No.1 (2011.5.11) を 一 部 改 変)

信書とは、「特定の受取人に対し、差出人の意思を表示し、又は事実を通知する文書」と、郵便法に定義されています。

「特定の受取人」とは、差出人がその意思の表示又は事実の通知を受ける者として特に定めた者のことです。文書自体に受取人の記載がないものであっても、その内容から、封皮に記載された受取人にあてたものであることが分かる場合には「特定の受取人」にあてたものとなります。また、受取人は個人に限定されるものではなく、会社、団体、組合等も含まれ、一人であっても複数人であっても具体的に定まっていればよいのです。

「差出人の意思を表示し、又は事実を通知する」とは、差出人の考えや思いを表し、又は現実に起こり、若しくは存在する事柄等の事実を伝えることです。「差出人の意思を表示し、又は事実を通知する」とは、差出人の考えや思いを表し、又は現実に起こり、若しくは存在する事柄等の事実を伝えることです。

同一内容で大量に作成された文書を個々の受取人に対して送付する場合であっても、内容となる文書が特定の受取人に対して意思を表示し、又は事実を通知するものであれば、信書に該当します。

「文書」とは、文字、記号、符号等、人の知覚によって認識することができる情報が記載された紙等のことです。文書の記載手段は、手書きに限らず、ワープロ等によるものでもよく、また、文書を記載する素材は、紙だけでなく、木片、プラスチック、ビニール等でもよいのです。なお、電磁的に記録されたCD等は、そこに記録された情報が人の知覚によって認識することができないものであるので、信書には該当しません。

添え状 ・ 送り状 (No.2 (2011.5.11) を 一 部 改 変)

郵便法第④条第③項には、`運送営業者、その代表者又はその代理人その他の従業者は、その運送方法により他人のために信書の送達をしてはならない。ただし、貨物に添付する無封の添え状又は送り状は、この限りでない」と規定されています。

この規定は、添状・送り状が受取人や運送営業者にとって貨物の点検等を行う場合に有益な文書であり、貨物を送付する際に添付されることが必要と認められることから設けられたものです。したがって、添え状・送り状は、貨物という送付の主体があって、その送付に関する事項が記載された文書が従として添えられる場合に限られるものです。

# 「添え状」とは

送付される貨物の目録や性質、使用方法等を説明する文書及び当該貨物の送付と密接に関連した次に掲げる簡単な通信文で当該貨物に従として添えられるもののことです。

- ア 貨物の送付に関して添えられるその処理に関する簡単な通信文
- イ 貨物の送付目的を示す簡単な通信文
- ウ 貨物の授受又は代金に関する簡単な通信文
- エ その他貨物に従として添えられる簡単な通信文であって、上記アから工までに掲げる事項に類するもの

# 「送り状」とは

貨物を送付したことを通知する案内書のことです。具体的には、送付される貨物の種類、重量、容積、荷造りの種類、個数、記号、代価、受取人並びに差出人の住所及び氏名等当該貨物の送付に関する事項が必要に応じて記載されたもののことです。

つまり、添え状・送り状は、信書に該当する文書ですが、例外的に、貨物を送付する際にこれらを添付してゆうパック等で送付することができるものです。

# 【質問】

お客様(不動産会社)から、マンションの売却を求めるダイレクトメール(文書)をゆうメールで送付することができるかというお問い合わせがありました。文頭に`〇〇マンションをご所有の皆様へ」と記載してありますが、これは「特定の受取人」に宛てた文書ではないため、信書に該当しないとしてゆうメールで送付できると考えてよいでしょうか。

# (文書)

00マンションをご所有の皆様へ

OOマンショソの購入を希望している方がいます。売却をお考えの方は是非ご検討ください。購入希望の方に紹介し、ご案内の調整も可能です。査定は無料で行っていますので、是非一度ご連絡下さい。

E 证5

△△不動産株式会社

東京都世田谷区1-2-3

電話 03--

担当：XX

この文書は、「〇〇マンションをご所有の皆様へ」という文言から、〇〇マンションの所有者という特定の受取人に対し、売却の検討を案内するなどという差出人の意思を表示し、又は事実を通知する文書であるため、信書に該当します。したがって、この文書をゆうメールで送付することはできません。受取人を特定することとなる文頭に記載される文言の例として、「〇△様」、「△ロ業生皆様」、「△〇購読者の皆様」、「〇口大学の卒業生の皆様」などがあります。

ー方、よく使用される「日頃ご利用いただきありがとうございます」という文言は、実際の利用の有無にかかわらず、商取引上の慣用語として使用されていますので、このような文言のみでは、商品の購入等利用関係があることを示す文言には該当しません。

お客さま(自動車ディーラー)から、「自動車検査証」(以下「車検証」といいます。)を、自動車の検査を受けた自動車の使用者(以下「使用者」といいます。)にゆうメールで送付することできるかとのお問い合わせがありました。車検証は、陸運局等から交付されるものですが、今回、車検証を送付する者は地方運輸局ではなく自動車ディーラーですので、単に貨物の送付に該当するとしてゆうメールで送付することができるということでよいでしょうか。

# 【回答】

この場合は、信書の送達に該当します。まず、車検証は、陸運局等が自動車の所有者に対して、登録された自動車が保安基準に適合していること及び記載された所有者が所有権を有しているという事実を通知し、又は意思を表示する文書であり、信書に該当します。

陸運局等が交付する車検証を、自動車ディーラーを介して使用者に送付する場合であっても、車検証は陸運局等から使用者に対して事実を通知(保安基準に適合していると証明)することに変わりはありません。よって、使用者という特定の受取人に届くまで、信書の送達に該当します。

したがって、この車検証を自動車ディーラーから使用者にゆうメールで送付することはできません。一方、自動車の所有者が受領した後においては、その車検証による事実の通知や意思の表示が既になされた後であるため、その原本もコピーも、信書に該当しません。

# 【質問】

お客様(○○商事)から、3件の顧客宛の請求書(文書)を、お客さまの本社から支店宛にゆうメールで送付することができるかというお問い合わせがありました。なお、これらの請求書は、支店から顧客に送付されるとのことです。請求書は、信書に該当する文書ですので、ゆうメールで送付することができないということでよろしいでしょうか。

# 【回答】

請求書は、代金を請求するという意思を表示し、又は事実を通知する文書ですので、差出人から特定の受取人にその内容を伝えるために送付する場合は、信書に該当します。本件の場合、願客宛の請求書を本社で一括して作成し、支店にまとめて送付する場合(①)は、これにより会社が意恐を表示し、又は事実を通知するものではないことから、信書の送達に該当しません。(その文書によって、〇〇商事が意思を表示し、又は事実を通知するのは、支店から顧客に交付する時点（②）です。)したがって、この文書を本社から支店宛に送付する際は、ゆうメールで送付することができます。
