# Cursor @Docs 登録用URLリスト

このファイルは、Cursor の `@Docs` 機能に登録するURLの一覧です。
各URLをコピーして、Cursor Settings > Features > Docs から追加してください。

---

## 登録手順

1. `Cmd + ,` でCursor Settingsを開く
2. **Features** タブをクリック
3. **Docs** セクションを見つける
4. **"Add new doc"** または **"+"** ボタンをクリック
5. 下記のURLを1つずつ貼り付けて追加

## ✅ 登録完了後の確認

登録が完了すると、Cursor AIチャット内で以下のように使用できます：

1. **`@Docs`** とタイプして全ドキュメントを参照
2. 特定のドキュメントを選択して質問
3. AIが最新の仕様を理解した上で回答

---

## 📌 Tips

- **優先度の高いもの**から登録することをお勧めします：
  1. Claude Agent SDK (プロジェクトの核心)
  2. Vercel AI SDK (UI統合の基盤)
  3. Next.js (フレームワーク仕様)
  4. 残りのライブラリ

- **インデックス化**には数分かかる場合があります
- 登録後は、チャット時に具体的なライブラリ名を言及すると精度が上がります

---

## 🔄 定期的な更新

これらのドキュメントは定期的に更新されるため、以下のタイミングで再登録を検討してください：

- メジャーバージョンアップ時
- 使用しているAPIの仕様変更時
- 新しいライブラリを追加する時
